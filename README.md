## Installation

To install the Matcha application, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/htuheind/matcha.git
   ```
2. Navigate to the project directory:
   ```bash
   cd matcha
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage

To run the application, use the following command:
```bash
npm start
```
